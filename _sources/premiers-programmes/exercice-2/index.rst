==========

..  tabbed:: first-programs-exercise-2

    ..  tab:: Donnée

        Le serpent doit se déplacer selon l'illustration. La fonction
        ``penDown()`` permet de faire en sorte que le serpent laisse une trace
        derrière lui. Cette trace n'est cependant visible que si le serpent ne
        la recouvre pas. La commande ``penDown()`` doit précéder les commandes
        de mouvement dans votre programme.
