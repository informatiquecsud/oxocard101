

Premiers programmes : les séquences
:::::::::::::::::::::::::::::::::::

.. admonition:: Objectifs d'apprentissage
   :class: info

   À l'issue de cette section, vous pourrez cocher les objectifs
   d'apprentissage suivants :

   *  Je sais que les programmes informatiques sont constitués de
      séquences d'instructions
   
   *  Je sais expliquer ce qu'est une séquence d'instructions
   
   *  Je suis capable d'écrire un programme simple constitué d'une
      séquence d'instructions
   
   *  Je suis capable d'utiliser le module ``oxosnake`` pour
      afficher un serpent sur la matrice LED de mon Oxocard.
   
   *  Je connais les commandes ``forward()``, ``left()`` et
      ``right()`` permettant de déplacer le serpent sur la matrice LED
      et je comprends leur effet sur le serpent.
