Répétitions avec ``repeat``
:::::::::::::::::::::::::::


.. admonition:: Objectifs d'apprentissage
   :class: info

   À l'issue de cette section, vous pourrez cocher les objectifs
   d'apprentissage suivants :

   *    Je sais que les ordinateurs sont très doués pour effectuer des
        tâches répétitives et rébarbatives
   
   *    Je sais ce qu'est un bloc d'instructions et je suis capable de
        créer un bloc d'instructions indenté de manière correcte et
        cohérente
   
   *    Je sais utiliser le mot-clé ``repeat`` pour répéter un bloc de
        code un nombre déterminé de fois
   
   *    Je suis conscient que le mot-clé ``repeat`` ne fait pas partie
        du langage Python standard et que je verrai plus tard comment
        créer des boucles avec les structures offertes par le Python
        standard.
   
   *    Je sais utiliser les boucles pour éviter de copier-coller
        inutilement du code identique
